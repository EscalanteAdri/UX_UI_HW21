Design Thinking

Double Diamond

The double diamond breaks down the stages of design thinking: Empathy, Define, Ideate, Prototype, and Test. If our goal is to improve user experience, it is important to start with Empathy. Without taking time to know and connect with the community, it is impossible to build a platform that addresses user's concerns and fufills users wants. Defining the problem includes clarifying the problem, defining the goals, and identifing the root cause. Ideate is where we create solutions, evaluate solutions, and select solutions to implement based on the users needs and wants. Prototype phase is where the first three steps come to life in a Low-fi proof of concept then High-Fi final product. Finally we have Testing, although it is technically the last stage, it is not the final step. Testing allows for UX designers to better empathize with the community, and the double diamond starts again. Design Thinking is non-linear, any only outlines the stages in designing.
